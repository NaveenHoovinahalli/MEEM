Terminologie

Appareil

Un smartphone ou une tablette compatible avec le câble MEEM

Appareil maître

L'appareil auquel le câble MEEM est branché (et sur lequel l'appli MEEM est lancée) est l'appareil maître. À tout moment, un seul appareil maître est accepté sur un réseau WiFi.

Appareils esclaves

Le ou les appareils sur lesquels l'application MEEM est installée, mais au(x)quel(s) le câble MEEM n'est pas branché sont les appareils esclaves sur un réseau MEEM.
