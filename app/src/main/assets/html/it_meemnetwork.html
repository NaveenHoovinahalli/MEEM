MEEM NETWORK

The MEEM Network allows you to backup up to 3 devices wirelessly to the MEEM cable as long as one of those devices has the MEEM cable plugged into it. For a full instruction guide on how the MEEM Network works and how to set it up please see the MEEM Network User Guide which can be found in the Help area of the MEEM app and at on our website on the Help page and at the bottom under DOWNLOADS at https://www.meemmemory.com/help/
