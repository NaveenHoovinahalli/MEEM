Automatische Sicherung

Nach der ersten Sicherung sichert MEEM jedes Mal Ihre Daten nach einem Countdown von 20 Sekunden automatisch, wenn Sie den Akku Ihres Smartphones damit aufladen.

Wenn sich die Inhalte auf Ihrem Gerät mehr als 5 % geändert haben, wird Sie MEEM automatisch darauf hinweisen. Falls Sie viele Informationen auf Ihren Gerät gelöscht haben, zum Beispiel nach Zurücksetzen auf Werkseinstellung (wodurch alle Daten auf Ihrem Gerät gelöscht werden), können Sie dadurch entscheiden, ob Sie die gleichen Daten auch auf Ihrem MEEM löschen möchten.

Wenn Sie die Sicherung abbrechen und die Daten von Ihrem MEEM auf Ihr Telefon wiederherstellen möchten, drücken Sie oben auf dem Bildschirm auf „Abbrechen“ und folgen den Anweisungen weiter unten.
